# Архитектура

:::note
Архитектура проекта основана на REST API, чётко разделённом на следующие домены: профили пользователей, настройки уведомлений, рекомендации, поиск и ресурсы, управление вакансиями (HR), а также аналитика и отчётность. На верхнем уровне находятся клиенты (веб-приложения, мобильные приложения, сторонние интеграторы), взаимодействующие с Backend API.

Backend представлен совокупностью функциональных подсистем:

User & Notification Service: Управляет профилем пользователя и настройками уведомлений.
Recommendation Service: Предоставляет рекомендации по вакансиям, курсам и карьерным путям.
Search Service: Реализует поиск по заданным критериям.
Resource Service: Отвечает за предоставление информации о конкретных ресурсах.
HR Vacancy Service: Осуществляет операции по созданию, обновлению и удалению вакансий, а также управляет их статусами.
Analytics Service: Обеспечивает анализ данных, формирование отчётов и получение трендов.
:::
