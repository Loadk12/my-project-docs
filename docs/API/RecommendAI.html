Получение рекомендаций

type
string
Enum: "vacancies" "courses" "career-paths"

Тип рекомендаций (vacancies, courses, career-paths)

integer
Default: 1

Номер страницы для пагинации

pageSize
integer
Default: 20

Количество рекомендаций на странице
